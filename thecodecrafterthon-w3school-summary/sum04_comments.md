# GO Comments
A Comment is a text to be ignored by the compiler upon execution. Comments can be used to explain line of code. Comments can also be used to prevent a line of code from execution.
GO supports single line or multi-line comments

### GO Single Line Comments
Single line comments start with two forward slashes `//`
***Example:***
`// This statement will print Hello World`
`fmt.Println("Hello World")`

### GO Multi Line COomments
Multi line comments start with `/*` and ends with `*/`
***Example:***
`/* The statement below will print Hello World`
`to the screen, and it is amazing*/`
`fmt.Println("Hello World")`

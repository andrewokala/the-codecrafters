# HOW TO RUN THE CALCULATOR

## Step 1:
In your CLI e.g Bash or Command Prompt after cloning the repository type cd thecodecrafterthon-day1, when you're in the folder type go run calculator.go to run the program

## Step 2:
You'll be prompted to input your First Input Number and also your Second Input Number for calculation.

## Step 3:
After the program recieves your inputed numbers you'll be prompted the operation you want to perform e.g (
    Addition
    Subtraction
    Multiplication
    Division
)
And it will output the result in the CLI. You can perform multiple calculations as long as you don't opt for exit
